
# TaskFlow Lite

TaskFlow Lite is a browser-based task management application built using vanilla JavaScript.

## Features
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Complete
- Data persistence using localStorage
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Run Project
1. Download the project
2. Open folder in VS Code
3. Open index.html in browser
